This function is designed to check the existence of a file and answers the question: "The file does not exist ??".

The function can also answer the questions: "Will the file be closed?" and "Can't read the file?"



Эта функция придумана для проверки на наличие файла и отвечает на вопрос: "Файл же не существует?".

Функция также может ответить на вопросы: "Файл закроется?" и "Файл же нельзя читать?".

Если вам нужно узнать ответ на вопрос:
  "Файл же не существует?", то вы можете результат от CheckFile() подставить в функцию bool(). False и None - Нет, True - Да. 
  "Файл же нельзя читать?", то ответ будет в типе возвращаемого результата None('NoneType') - Да, 'bool' - Нет.
  "Файл закроется?", то воспользуйтесь: try, except(Исполниться только если в try есть Исключение).

Функция может понять и вернуть 4 события:
 Файла нет(True) или файл есть и читается(False) а также файл есть, но не читается(None), Файл не закрылся.


Кроме 3 событий(от return) также есть и 1 Исключение(TheFileDidntClose - от raise) - Говорит что файл не закрывается.
Арсенал: 2 bool значения. 1 NoneType и 1 Exception.
События:
 Файл существует: None - Файл не прочитался. False - Файл прочитался.
 Файл не существует: True.
 Файл не закрывается: TheFileDidntClose.

Если вам часть функционала будет без надобности, вы вправе убрать его (Если вы понимаете что делаите).
